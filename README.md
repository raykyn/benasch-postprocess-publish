# BeNASch-Postprocessing

## Context
This repository contains code for postprocessing documents from [INCEpTION](https://inception-project.github.io/), adding information and transforming them into [BeNASch](https://dhbern.github.io/BeNASch/)-XML. It (will) also provide code to further transform these XML to formats such as column-format for maschine-learning and TEI.

Please find more information in the (german-language) [Guide](https://dhbern.github.io/BeNASch/5_anno_guide/5_3_postprocessing.html).

## Usage

**Install Python**
Python 3.8 or newer

**Clone the Repository**
```bash
git clone https://github.com/raykyn/benasch-postprocess-publish.git
cd benasch-postprocess-publish
```

**Install dependencies**
pip install -r requirements.txt

**Run**
Please see the Guide for more info on how to set up and run the scripts.
